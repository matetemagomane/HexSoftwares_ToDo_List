# HexSoftwares To-Do List App

A simple and interactive to-do list application built with HTML, CSS, and JavaScript as part of the Hex Software Internship Program.

## Project Description
This is a fully functional to-do list application that allows users to add, complete, and delete tasks. All tasks are stored locally in the browser's local storage, ensuring data persistence even after closing the browser.

## Features
- Add new tasks with a clean input interface
- Mark tasks as completed with checkboxes
- Delete individual tasks
- Task counter showing total and completed tasks
- Local storage integration for data persistence
- Responsive design for mobile and desktop
- Modern gradient UI with smooth animations
- Empty state message when no tasks exist

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How to Use
1. Type your task in the input field
2. Click "Add" button or press Enter to add the task
3. Check the checkbox to mark a task as completed
4. Click "Delete" to remove a task
5. Your tasks are automatically saved and will persist even after closing the browser

## Installation
1. Clone this repository
2. Open `index.html` in your browser
3. Start managing your tasks!
